This workshop requires an AWS account with Bedrock model access, a LangSmith account on the AWS-region instance, and a development environment with Python 3.11+.

You have two options:

- [At an AWS Event](/00-prerequisites/01-at-an-aws-event/) - instructor-led session with a pre-provisioned account
- [Self-Paced Setup](/00-prerequisites/02-self-paced/) - running in your own AWS environment
